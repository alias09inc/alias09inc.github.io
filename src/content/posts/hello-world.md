こんにちは！このブログはAstroのContent Collectionsを使ってMarkdownから記事を生成しています。

## このブログでできること

- `src/content/posts` にMarkdownを追加するだけで記事が増える
- Astroが静的HTMLにコンパイルしてGitHub Pagesへデプロイ
- シンプルで高速な読み込み体験

## 次にやること

1. `title` や `description` などのフロントマターを自分の内容に変更する
2. 画像やコードブロックを追加して記事を書き進める
3. デプロイして公開URLから記事を確認する

```bash
npm run dev
```

ローカルで確認してから公開しましょう！
